# 代購作業台

代購作業台是一個手機優先的 PWA 工具，用來匯入賣貨便訂單 Excel、建立代購團、追蹤採購進度，並估算報價、成本與獲利。

資料會存在目前使用的手機或瀏覽器本機。只要沒有清除瀏覽器資料、移除 App，或更換裝置，資料基本上會保留在原本的手機上。

## 可以做什麼

- 建立多個代購團，分開管理不同批次訂單。
- 匯入賣貨便匯出的 `.xlsx` 訂單檔。
- 依商品名稱彙整訂購數量、訂單筆數與單價。
- 再次匯入同一團的新 Excel，更新商品數量並保留已填寫的資料。
- 輸入對客報價匯率、實際成本匯率與日幣原價。
- 批次回推尚未設定的日幣原價。
- 追蹤已代購數量、剩餘數量、缺成本與低利潤品項。
- 查看預估營收、成本、獲利與利潤率。
- 匯出資料備份 JSON。
- 自訂手機版上方工作區名稱與 Topbar 底圖。

## 手機安裝

請使用正式 `https` 網址開啟此工具，才能正常啟用 PWA 安裝與離線快取。

### iPhone

1. 用 Safari 開啟網站。
2. 點選 Safari 下方的「分享」。
3. 選擇「加入主畫面」。
4. 回到主畫面後，就可以像一般 App 一樣開啟。

### Android

1. 用 Chrome 開啟網站。
2. 若瀏覽器出現安裝提示，可直接點選安裝。
3. 也可以從 Chrome 選單中選擇「新增至主畫面」或「安裝應用程式」。

## 基本使用流程

1. 開啟代購作業台。
2. 點選「新增代購團」。
3. 選擇賣貨便匯出的 `.xlsx` 訂單檔。
4. 視需要修改代購團名稱。
5. 輸入對客報價匯率與實際成本匯率。
6. 針對商品輸入日幣原價，或使用「全部回推日幣」批次估算。
7. 採購後填寫已代購數量。
8. 後續訂單有變動時，進入該團並點選「更新此團訂單」，再匯入新的 Excel。

## 手機匯入賣貨便訂單

手機版也可以匯入賣貨便訂單，只是流程會比電腦多幾步：

1. 用手機瀏覽器登入賣貨便。
2. 到訂單頁匯出訂單 Excel。
3. 回到代購作業台。
4. 點選新增或更新代購團。
5. 選擇剛剛匯出的 Excel 檔案匯入。

如果只是想試用，可以先建立測試用的代購團玩玩看；正式使用前建議先確認匯入檔案與資料備份流程。

## 資料保存與備份

目前資料保存在同一台裝置、同一個瀏覽器的 `localStorage`。

請注意：

- 換手機、換瀏覽器或清除瀏覽器資料後，資料不會自動同步。
- iPhone 或 Android 若移除主畫面 App、清除網站資料，可能會連同資料一起刪除。
- 若要長期保存，請定期使用「備份資料」匯出 JSON。
- 備份檔請另外存到雲端硬碟、電腦或其他安全位置。

## 畫面更新注意事項

此工具使用 PWA 快取，正常情況下重新整理會取得最新版本。

如果剛更新後畫面仍像舊版，或顏色、排版看起來不一致，可以嘗試：

- 重新整理頁面。
- 關閉 App 後重新開啟。
- 在瀏覽器中強制重新整理。
- 必要時清除該網站的快取後再開啟。

## 本機測試

如果要在電腦本機測試，可以在專案資料夾執行：

```powershell
python -m http.server 8080
```

然後用瀏覽器開啟：

```text
http://localhost:8080/
```

直接打開 `index.html` 也可以使用主要功能，但不會啟用 PWA 安裝與 service worker 快取。
